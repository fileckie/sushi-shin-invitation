# 鮨心 · 高端日式料理邀请函系统

> 专为高端日料店打造的私宴邀请函系统，融合日式极简美学与一期一会的精神。

## 品牌概念

**鮨心 (Sushi Shin)** - 银座高端寿司店
- 传承江户前寿司精髓
- 店主修业于银座久兵卫
- 仅8个座位，完全预约制
- 米其林星级餐厅

## 设计特色

### 日式极简美学
- **配色**: 和纸白 `#F7F5F0` + 墨色 `#1A1A1A` + 朱色 `#C73E3A`
- **字体**: 日本明朝体 (Noto Serif JP)
- **留白**: 大量呼吸空间，日式排版
- **纹理**: 和纸质感背景
- **动效**: 墨染效果、竖排文字

### 核心功能
1. **招待状作成** - 日式邀请函生成
2. **お品書き** - 对外菜单（给客人）
3. **仕込み帖** - 对内员工卡
4. **一期一会の記録** - 宴请管理

## 安装运行

```bash
npm install
npx prisma db push
npm run dev
```

访问 http://localhost:3000
